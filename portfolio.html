The Recipe Box

This full-stack app utilze a SQL database to store recipes that the user can view. Users can also search for recipes in using multiple different search parameters, add new recipes of their own, comment on recipes, email a recipe to someone and share recipes on social media. This application also has a "Favorites" recipe page that shows the most "liked" recipes. It is deployed on Heroku.

    The Burger App

    This application is a simple full-stack application using JavaScript, NodeJS, Express, MySQL, HTML5, CSS, and a small custom ORM. It lists burgers that can be "devoured". When the user clicks the "Devour it" button, the burger is then moved over onto the Devoured Burger list. The user may input new burgers in the form, which will then update the "Burgers to Devour" list. The burger information is all rendered through server-side rendering using express-handlebars.

    Weather Dash

    This application utilizes the OpenWeatherAPI to obtain information about the current and 5 day forecast for the location that a user searches for. Upon the user entering a town and selecting a state, and the current weather and 5 day forecast elements will be dynamically created and appended to the webpage to display the weather information for the location searched. The user's last search is saved in local storage so that on revisiting or reloading the page, the same request is made of the API and the current and 5 day weather forecast are displayed.

    Day Scheduler

    This is a small front-end project that utilizes local storage and Moment.js. The user can type in events for each hour and save them to local storage when the Save button is clicked. As time goes by, various style changes occur to demonstrate when events are past, currently happening, or upcoming. Once the user no longer needs the events stored, they may clear the schedule, which clears local storage.
